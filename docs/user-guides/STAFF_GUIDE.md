## Managing Your Queue

### View Queue Status

1. Select a queue from your dashboard
2. You'll see:
   - Current token being served
   - Number of waiting customers
   - Next token in line
   - Queue status (Active/Paused)

### Call Next Token

This is your primary action!

1. When ready for the next customer:
2. Click **"Call Next"**
3. The system will:
   - Call the next token in line
   - Notify the customer
   - Display the token number
   - Update everyone's position

**What happens:**
- Customer receives notification
- Token status changes to "CALLED"
- Display shows the token number
- Other customers see updated positions

### Serve Current Token

After helping the customer:

1. Click **"Mark as Served"**
2. Confirm the action
3. Token status changes to "SERVED"
4. You're ready for the next customer

### Handle Priority Tokens

Some tokens have priority (emergency, VIP):

1. Priority tokens appear at the top
2. Call them first
3. System automatically manages order

---

## Queue Controls

### Pause Queue

Use when taking a break:

1. Click **"Pause Queue"**
2. Confirm
3. No new tokens will be called
4. Waiting customers are notified

**When to pause:**
- Lunch break
- End of shift
- System maintenance
- Emergency situations

### Resume Queue

When ready to continue:

1. Click **"Resume Queue"**
2. Queue continues from where it was paused
3. Customers are notified

---

## Handling Special Situations

### Customer No-Show

If a customer doesn't appear:

1. Wait 2-3 minutes
2. Click **"Call Next"** to skip
3. Skipped token goes to the end
4. Customer can still be served later

### Customer Cancellation

If customer cancels in person:

1. Find their token
2. Click **"Cancel Token"**
3. Confirm cancellation
4. Token is removed from queue

### Walk-in Customers

For customers without a token:

1. Click **"Create Walk-in Token"**
2. Enter customer details (optional)
3. Token is created and called immediately

## Understanding Token Status

| Status | Meaning | Your Action |
|--------|---------|-------------|
| **PENDING** | Waiting in queue | Will be called automatically |
| **CALLED** | Customer's turn | Serve the customer |
| **SERVED** | Service complete | Mark as served |
| **CANCELLED** | Customer cancelled | No action needed |
| **NO_SHOW** | Customer didn't appear | Call next |

## Troubleshooting

### Can't Call Next Token

**Possible Reasons:**
- Queue is paused
- No tokens waiting
- System error

**Solution:**
- Resume queue if paused
- Wait for customers to join
- Contact admin if error persists

### Token Not Updating

**Check:**
- Internet connection
- Browser refresh
- System status

**Solution:**
- Refresh the page
- Check connection
- Report to admin

### Customer Says They Weren't Notified

**Verify:**
- Token status
- Customer's app status
- Notification settings

**Solution:**
- Check token in system
- Ask customer to refresh
- Manually notify customer
